## What It Does

This Windows-focused utility for Dead by Daylight is aimed at automating match flow and helping with resource farming. It reduces the need for repeated manual grinding by carrying out in-game actions on your behalf, which can make bloodpoint accumulation faster and less hands-on. The trainer also includes ESP-based visual aids that surface the positions of players and objectives on the map.

The latest build emphasizes more dependable automation behavior and broader ESP wallhack coverage. That means smoother farming loops, steadier match handling, and improved recognition of in-game events. It runs as a standalone script and connects with the game client using standard input simulation plus memory reading methods.
